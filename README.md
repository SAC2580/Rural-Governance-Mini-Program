# Rural-Governance-Mini-Program
乡村治理小程序
基于微信小程序的乡村治理系统的设计与实现

技术说明：
用户前端：微信小程序原生框架
管理前端：Vue.js（RuoYi框架）
服务端：基于Java的SpringBoot框架
数据库：MySQL

功能说明：
用户端：
1. 首页
    轮播图片
    乡村快讯（乡村快讯/春耕播种/政策服务/预警信息分类Tab）
2. 通知公告
3. 活动列表
    活动分类（文化活动/志愿服务/技能培训/体育健身/公益慈善/村民议事/节庆活动/环境整治）
    进行中/已结束
    活动详情
4. 活动报名
    填写姓名、手机号、备注
    生成报名编号
    报名状态（报名成功/取消报名/签到成功）
5. 活动搜索
6. 村民论坛
    分类浏览（招工就业/相亲脱单）
    发布帖子
    上传图片
    帖子详情
7. 论坛评论
8. 我的帖子
    审核状态（审核中/通过/不通过）
9. 乡村特产
    商品分类（农副产品/时令果蔬/手工制品/特色美食/传统工艺）
    商品列表
    商品详情
10. 乡村服务
    服务分类（户籍办理/婚姻登记/土地管理/证明开具/法律援助）
    服务详情（地点、流程、联系人、电话、工作时间）
11. 服务预约
    填写姓名、电话、身份证
    选择日期时间
    生成预约编号
    预约状态（预约审核/预约成功/预约失败/取消预约）
12. 我的报名
13. 我的预约
14. 数据统计
    ECharts月趋势图
    活动分类分布图
15. 村域点位
    地图模式/列表模式切换
    附近村落导航
    村落详情（人口、产业、特色、交通路线）
16. 微信登录
17. 个人中心
    用户信息展示
    退出登录
18. 关于系统
19. 联系我们

管理端：
1. 首页统计分析
2. 用户信息管理
3. 轮播图片管理
4. 乡村快讯管理
    分类管理（乡村快讯/春耕播种/政策服务/预警信息）
5. 系统通知管理
6. 活动类别管理
7. 活动信息管理
8. 活动报名管理
    签到状态
9. 乡村特产管理
10. 论坛帖子管理
    审核通过/不通过
11. 论坛评论管理
12. 乡村服务管理
13. 服务预约管理
    预约审核

项目创新点：
1. 村域点位支持地图/列表双模式切换，实现附近村落快速定位与导航
2. 活动推荐融合协同过滤、内容推荐与热度推荐三种算法，精准匹配村民兴趣
3. 乡村服务覆盖户籍办理、婚姻登记等政务服务，预约审核流程规范化
4. 乡村特产商城按农副产品/时令果蔬等五大分类，促进农产品线上销售
5. 招工就业与相亲脱单分类论坛，针对性满足乡村青年核心社交需求
6. ECharts数据统计提供活动月趋势与分类分布可视化，辅助乡村管理决策
7. 活动分类涵盖八大类型（文化/志愿/培训/体育/公益/议事/节庆/环境），丰富乡村文化生活
